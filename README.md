# Personal Blog

This is my personal blog built with React and Vite.

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone git@github-personal:amarqueze/amarqueze.github.io.git
cd amarqueze.github.io
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

The app will be available at:

```text
http://localhost:5173
```

## 📦 Build for production

```bash
npm run build
```

## 🔍 Preview production build

```bash
npm run preview
```

## 🛠 Tech Stack

* React
* Vite
* JavaScript
* CSS
